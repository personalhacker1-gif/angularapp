Pipeline Process Architecture

Step 1: Jenkinsfile aur Environment Config

Angular project ke andar Jenkinsfile banayi, node20 configuration load kiya, aur Jenkins Credentials mein VERCEL_TOKEN ko secure kiya.

Step 2: Local Vercel Linkage

Terminal par Vercel CLI se project link kiya. Generated .vercel/project.json file ko Git par push kiya taaki Jenkins automated deploy kar sake.

Step 3: GitHub Webhook Integration

GitHub Repository ki settings mein Jenkins Payload URL configure kiya, jisse local code push hote hi Jenkins auto-trigger ho jaye.

Step 4: Fully Automated CI/CD Execution

Webhook trigger hote hi Jenkins ne automatic npm packages install kiye, build banaya, aur bina manual intervention ke --yes flag se Vercel par live kar diya.
